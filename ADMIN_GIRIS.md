# Admin Giriş Bilgileri

## Admin Paneline Giriş

Admin paneline giriş yapmak için aşağıdaki bilgileri kullanın:

**URL:** `http://localhost:3000/login`

**Email:** `admin@atolyem.local`  
**Şifre:** `admin123`

## Seller Giriş Bilgileri

Seller paneline giriş yapmak için:

**Email:** `artist@example.com`  
**Şifre:** `seller123`

## Notlar

- Bu bilgiler `prisma/seed.ts` dosyasında tanımlanmıştır
- İlk çalıştırmada `npm run prisma:seed` komutunu çalıştırmanız gerekebilir
- Admin paneli: `/admin`
- Seller paneli: `/seller`
- Katalog: `/catalog`
